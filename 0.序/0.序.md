# 序章：末日与曙光
## 危机
### 2033年1月1日 7:00
天空不再蔚蓝，而是被一片诡异的且刺眼的光芒笼罩。牧夫座空洞中出现的超临界黑洞，像一只贪婪的巨兽，无情地吞噬着周围的一切。地球，这个曾经孕育了无数生命的蓝色星球，如今正面临着前所未有的灾难。人类伤亡惨重，90%的人口被巨大的引力撕成碎片，剩下的幸存者们面临着一个残酷的现实——地球只剩下30%的区域相对安全。

在纽约，曾经繁华的街道如今变得空荡荡的。高楼大厦之间，偶尔能听到几声凄厉的警报声，像是在为这个末日世界奏响哀歌。幸存的人们脸上写满了惊恐与绝望，他们拖着沉重的行李，带着仅有的希望，朝着那些所谓的“安全区”艰难前行。

在一处临时搭建的避难所里，一位年迈的老人坐在角落，眼神空洞地望着窗外。他的手中紧紧握着一张泛黄的照片，那是他年轻时与家人的合影。如今，他的家人早已在这场灾难中离他而去，只剩下他孤零零地活在这个世界上。泪水在他的眼眶中打转，但他却无力哭泣。他的心中充满了对过去的怀念和对未来的恐惧。

“我们必须得走了，这里不再安全。”一位年轻的救援队员走进避难所，大声地对众人说道。他的声音中透露出一丝疲惫，但更多的是坚定。救援队员们已经连续工作了数日，他们不断地在废墟中搜寻幸存者，将他们带到相对安全的地方。然而，随着黑洞的不断扩张，所谓的“安全区”也在逐渐缩小。

幸存的人们纷纷站起身，拿起自己的行李，跟随着救援队员的脚步，朝着未知的方向走去。他们中有的是失去父母的孩子，有的是失去伴侣的老人，还有的是失去家园的普通市民。他们的脸上都带着迷茫和不安，不知道前方等待他们的是什么。

在撤离的队伍中，有一位年轻的母亲紧紧抱着自己的孩子。她的孩子只有两三岁，还不明白这个世界发生了什么，只是好奇地四处张望。年轻的母亲心中充满了自责和愧疚，她觉得自己没有能力保护好自己的孩子，让他在这个末日世界中受苦。但她又不得不坚强起来，为了孩子，她必须继续前行。

“妈妈，我害怕。”孩子突然开口，声音带着一丝颤抖。

年轻的母亲低下头，温柔地抚摸着孩子的头发：“不怕，宝贝，妈妈会一直陪着你。我们会找到一个安全的地方，然后一切都会好起来的。”

她的话语中充满了坚定和希望，尽管她自己也不知道未来会怎样。但她知道，只要她还活着，就要为孩子撑起一片天空。

队伍继续前行，他们穿过了废墟，穿过了黑暗，朝着那微弱的希望之光走去。天空中，黑洞的光芒近乎掩盖了一切，仿佛在嘲笑着人类生命之火的微弱。

## 抉择
### 2033年1月1日 18:30
宇航员杰克森站在临时避难所的门口，望着眼前这群疲惫又绝望的幸存者，心中五味杂陈。他是人类历史上第一个前往外星系的人，曾经在浩瀚的宇宙中穿梭，探索未知的星球。然而，如今他却不得不面对一个艰难的抉择——带着这些幸存者逃离地球，但火箭的负载有限，不是每个人都能登上这艘希望之舟。

杰克森的脸上刻满了岁月的痕迹，他的眼神中透露出一种坚毅和无奈。他深知自己肩负着巨大的责任，每一个决定都可能改变无数人的命运。他深吸一口气，试图平复内心的波动，然后缓缓开口：“各位，我知道你们都渴望逃离这个末日之地，我也希望能带着大家一起去寻找新的家园。但是，我们必须面对一个残酷的现实——火箭的负载有限，我们不能带走所有人。”

人群中顿时爆发出一阵骚动，幸存者们的眼神中充满了惊恐和不安。一位年轻的母亲紧紧抱着自己的孩子，泪水在眼眶中打转：“求求你，宇航员先生，我的孩子还小，他什么都不知道。请让我们登上火箭，给我们一个生存的机会。”

杰克森的心被深深刺痛，他多么希望自己有能力满足这位母亲的请求，但现实却是如此残酷。他轻轻摇头：“我理解你们的感受，但这是无法改变的事实。我们必须做出选择，只有最有可能在新环境中生存下来的人才有资格登上火箭。”

他的话语如同一记重锤，砸在了众人的心上。幸存者们开始互相争论，有的人愤怒地指责杰克森的无情，有的人则默默地流下了绝望的泪水。杰克森站在一旁，静静地听着他们的声音，心中充满了愧疚和无奈。

就在这时，一位年迈的老人缓缓站起身，他的眼神中透露出一种超脱和宁静：“年轻人，我知道你很难做出决定。但这是为了大家的生存，我们必须理解。我年纪大了，已经没有多少时间了。我的位置应该让给那些年轻人和孩子们，他们还有未来。”

老人的话语如同一股清泉，浇灭了人群中的愤怒和混乱。他缓缓走向杰克森，将手中的泛黄照片递给他：“这是我年轻时与家人的合影。我希望你能带着他们的记忆，去寻找一个新的世界。在那里，人类能够重新开始，不再有战争和灾难。”

杰克森接过照片，心中涌起一股暖流。他望着老人那布满皱纹的脸庞，眼中闪过一丝感激：“谢谢您，老人家。您的理解和支持，对我来说意义重大。”

老人微微一笑，转身走向人群。他开始劝说其他人，让他们理解这个艰难的决定。慢慢地，人群中的情绪开始平息，大家开始接受这个残酷的现实。

杰克森知道，他必须尽快做出选择。他开始在人群中穿梭，仔细观察每一个人。他寻找那些身体强壮、年轻有活力的人，以及那些有着特殊技能和知识的人。他知道，这些人将是新世界中的希望。

在选择的过程中，杰克森的心中充满了矛盾和挣扎。他不得不放弃一些人，而这些人可能就是他的朋友、战友，甚至是亲人。每一次的选择，都像是在割舍自己的血与肉。但他知道，这是他必须承担的责任，为了人类的未来，他必须做出这些艰难的决定。

终于，经过数小时的筛选，杰克森确定了登上火箭的人员名单。他看着这份名单，心中充满了沉重。他知道，这些人将肩负着人类的希望，去寻找一个新的家园。而那些未能登上火箭的人，将不得不留在这个末日之地，面对未知的命运。

夜幕降临，天空中的黑洞依旧散发着刺眼的光芒。杰克森带领着选中的幸存者，走向停在不远处的火箭。他们的眼神中充满了希望和期待，仿佛已经看到了新世界的曙光。

在火箭的入口处，杰克森停下脚步，回头望了一眼那些未能登上火箭的人。他们的眼神中充满了不舍和绝望，但更多的是对杰克森的理解和支持。杰克森心中一酸，他知道，他可能再也见不到这些人了。

“各位，保重。”杰克森低声说道，声音中带着一丝哽咽。

然后，他转身走进火箭，开始了人类历史上最伟大的迁徙之旅。

## 中转站
### 2033年2月1日 7:00
经过了一个月的煎熬，杰克森带领的幸存者们终于抵达了他曾经探索过的那颗半宜居星球。这颗星球位于距离地球数光年之外的星系，它有着类似地球的大气成分和重力环境，但气候条件更为恶劣，生态系统也远不如地球丰富。

火箭缓缓降落在星球表面，扬起一片尘土。杰克森站在观察窗前，望着外面那陌生而又熟悉的景象，心中百感交集。这颗星球的地表呈现出一片荒芜的黄色，偶尔能看到一些低矮的植被在风中摇曳。天空是一片灰蒙蒙的色调，没有地球上的蓝天白云，但至少没有黑洞那种令人绝望的光芒。

“我们到了。”杰克森的声音在火箭内部回荡，他的语气中带着一丝疲惫，但更多的是释然。幸存者们纷纷围拢过来，透过观察窗好奇地打量着这个新世界。他们的脸上写满了期待和不安，不知道这里将给他们带来什么样的生活。

杰克森转身面对众人，他的眼神中闪烁着坚定的光芒：“各位，我们终于到达了目的地。这颗星球虽然不是地球，但它将是我们的新家园。我知道，这里的生活将会充满挑战，但我们必须团结一心，共同面对。我相信，只要我们努力，就一定能够重建我们的文明。”

幸存者们被杰克森的话语所鼓舞，他们纷纷点头，眼中闪烁着希望的光芒。杰克森知道，他必须尽快让大家安定下来，开始新生活。他开始分配任务，让一部分人负责搭建临时住所，另一部分人去探索周围的环境，寻找可用的资源。
搭建临时住所

在杰克森的指挥下，幸存者们开始忙碌起来。他们利用火箭上携带的材料，开始搭建临时的住所。这些住所虽然简陋，但在恶劣的环境中，它们将是大家的避风港。

杰克森亲自参与到搭建工作中，他的动作熟练而有力。他的心中充满了责任感，他知道，这些住所的质量将直接影响到大家的生存。他不断地鼓励着身边的幸存者，让他们保持积极的心态。

“大家加油，我们很快就能有自己的家了。”杰克森大声说道，他的声音中充满了力量。

幸存者们被杰克森的激情所感染，他们纷纷加快了手中的工作。虽然大家的脸上都带着疲惫，但眼中却闪烁着希望的光芒。他们知道，只要大家齐心协力，就一定能够克服眼前的困难。

与此同时，另一部分幸存者在杰克森的副手艾米的带领下，开始探索周围的环境。他们带着必要的装备，小心翼翼地在星球表面行走，寻找可用的资源。

艾米是一位经验丰富的地质学家，她对这颗星球的地质结构有着深入的了解。她带领着队伍，仔细地观察着周围的地形和植被。她的目标是找到水源和食物，这是大家生存的关键。

“大家注意，这里的植被虽然稀少，但很可能隐藏着一些可食用的植物。”艾米提醒着队伍中的成员，她的声音中带着一丝严肃。

队员们纷纷点头，他们开始仔细地搜寻周围的植被。虽然这里的环境恶劣，但他们还是找到了一些类似地球上的野菜和果实。这些发现让大家的心情稍微放松了一些，他们知道，至少在食物方面，他们暂时不用担心了。

然而，就在大家忙碌的时候，一场突如其来的沙尘暴打破了平静。沙尘暴来势汹汹，遮天蔽日，让所有人都措手不及。杰克森立刻组织大家回到临时住所，躲避这场灾难。

沙尘暴中，杰克森站在住所的门口，望着外面那狂暴的自然力量，心中充满了无奈。他知道，这个新世界并不会轻易地接纳他们，他们必须时刻准备面对各种挑战。

“大家注意安全，沙尘暴很快就会过去的。”杰克森大声喊道，他的声音在风中显得有些微弱。

幸存者们紧紧地靠在一起，他们的心中充满了恐惧，但更多的是对杰克森的信任。他们知道，只要跟着杰克森，就一定能够度过难关。

终于，沙尘暴过去了，天空重新变得清澈。杰克森带领着大家走出临时住所，开始清理沙尘暴带来的破坏。虽然损失了一些物资，但大家的意志并没有被击垮。他们知道，这只是新世界给他们的一次考验，他们必须坚强地面对。

在接下来的日子里，杰克森和幸存者们开始了艰苦的重建工作。他们利用星球上的资源，逐渐改善了生活环境。虽然生活依然艰苦，但大家的心中充满了希望。

杰克森站在临时住所的屋顶上，望着远方那逐渐升起的太阳，心中充满了感慨。他知道，这个新世界虽然充满了挑战，但也是充满了希望。只要大家团结一心，就一定能够重建人类的文明。

“我们一定会成功的！”杰克森低声说道，他的声音中充满了坚定。

## 天文台
### 2033年2月10日 9:00
在这颗半宜居星球上，杰克森和幸存者们已经度过了近十天的艰苦时光。虽然环境恶劣，但他们逐渐适应了这里的生活节奏。然而，所有人都清楚，这颗星球只是一个中转站，他们需要找到一个更适宜长期居住的星球。为了实现这个目标，大家决定合力搭建一个天文台，以便更好地观察宇宙，寻找那个能够给予他们更长久安全时间的家园。

杰克森召集了所有幸存者，围坐在临时住所的中央。他的眼神中闪烁着坚定和希望，声音洪亮而有力：“各位，我们已经在这颗星球上生存了几天，但这里只是我们的中转站。我们需要一个更稳定的家园，一个可以让我们的后代继续繁衍的地方。为了找到这样的星球，我们需要搭建一个天文台，让我们能够更好地观察宇宙。”

幸存者们围坐在一起，聆听着杰克森的计划。他们的脸上写满了认真和期待，尽管知道前方的道路充满挑战，但为了生存和未来，他们愿意付出一切努力。

“搭建天文台需要我们所有人共同努力。我们需要收集材料，设计结构，还要确保它能够抵御这里的恶劣天气。”杰克森继续说道，他的声音中充满了信心。

艾米，作为地质学家，站起身来，补充道：“我建议我们先对这颗星球的地质结构进行更深入的研究，找到一个稳定且适合搭建天文台的地点。同时，我们还需要考虑如何利用当地的资源，比如岩石和土壤，来构建天文台的基础。”

杰克森点头表示赞同：“艾米说得对。我们需要充分利用这里的资源。此外，我们还需要一些技术专家来设计天文台的观测设备。虽然我们的设备有限，但只要我们齐心协力，就一定能够完成这个任务。”

在杰克森和艾米的领导下，幸存者们开始了紧张而有序的工作。一部分人负责收集搭建天文台所需的材料，另一部分人则跟随艾米进行地质勘探，寻找合适的搭建地点。

收集材料的队伍在星球表面四处搜寻，他们找到了一些坚硬的岩石和适合用来构建结构的土壤。虽然这些材料与地球上的有所不同，但在艾米的指导下，他们学会了如何利用这些资源。

“这些岩石虽然质地坚硬，但经过加工后可以用来构建天文台的基座。”艾米向队员们解释道，她的手指轻轻敲击着一块岩石，发出清脆的声音。

与此同时，地质勘探队伍在艾米的带领下，对星球的地质结构进行了详细的调查。他们发现了一个相对平坦且地质稳定的区域，这里远离沙尘暴的频繁侵袭，是搭建天文台的理想地点。

“这里地质稳定，土壤承载力强，非常适合搭建我们的天文台。”艾米兴奋地向杰克森汇报，她的脸上洋溢着成就感。

杰克森听后，心中充满了希望：“太好了，艾米。有了这个地点，我们的工作就成功了一半。接下来，我们需要尽快开始搭建工作。”

在确定了地点和收集了足够的材料后，幸存者们开始了天文台的搭建工作。尽管条件艰苦，但大家的热情高涨，每个人都为了这个共同的目标而努力。

杰克森和艾米负责监督整个搭建过程，确保每一个环节都符合设计要求。技术专家们则忙碌于安装观测设备，尽管设备简陋，但在他们的巧手中，逐渐形成了一套基本的观测系统。

“这个望远镜虽然不如地球上的先进，但足以让我们观察到周围的星系。”一位技术专家向杰克森展示着他们自制的望远镜，他的眼中闪烁着自豪的光芒。

杰克森点头称赞：“你们做得很好。有了这个望远镜，我们就能开始寻找那个理想的家园了。”

在接下来的几天里，幸存者们日夜奋战，终于完成了天文台的基本搭建。虽然它看起来简陋，但在大家的心中，它是通往新家园的希望之门。

天文台搭建完成后，杰克森和技术团队迫不及待地开始了首次观测。夜幕降临，星球的天空中繁星点点，虽然没有地球上的银河那般壮丽，但也别有一番风味。
